# clj-css

Parse, emit, and convert between various CSS file formats

## Usage

```
(parse-css css)
(emit-css css)
```

# CSS Formats

* CSS
* SCSS
* SASS

# Tests

```
lein test
```

## Dependencies

[CSS Parser](http://cssparser.sourceforge.net)
